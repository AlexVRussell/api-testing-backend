# api-testing-backend

## description
A lightweight backend service that stores and executes HTTP requests.

The goal of this project is to build a backend-only API client from scratch. The initial version focuses on one simple task: sending HTTP requests and returning the response.

As the project grows, features like request history, collections, authentication, analytics, and scheduling can be added incrementally.
```
- Create an HTTP request
- Execute the request
- Return the response
- Support common HTTP methods (GET, POST, PUT, DELETE)
```
## tech stack
```
- Java 21
- Spring Boot
- Spring Web
- Maven
```
